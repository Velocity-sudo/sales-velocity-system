# Prompt Adaptado: "Link Hub" de Alta Conversión para Johanna Cueva

## **Actúa como un CRO Specialist, Growth Marketer y Web Designer premium (mobile-first), especializado en “link-in-bio” pages de alta conversión para Instagram.**

**Contexto:**
Johanna Cueva es una creadora de contenido en Miami enfocada en **Dance, Wellness y Mindset**.
Su diferencial NO es la perfección técnica, sino el **SENTIR el baile**. Su comunidad son mujeres (principiantes a expertas) que vibran con la **Salsa** y buscan confianza.
Está en una fase de **Crecimiento Exponencial**. Busca marcas **conscientes** (Huella verde, alimentación limpia, moda sostenible, vitaminas) que quieran crecer con ella AHORA antes de que su caché suba.
Necesita una Landing Page estilo "Link-in-Bio" pero High-End, que funcione como un embudo de ventas para marcas y colaboraciones.

**Audiencia:**
1.  **Marcas Conscientes (B2B)**: Buscan autenticidad y valores alineados (Wellness, Eco-friendly). Necesitan ver el "Crecimiento" para sentir la urgencia de invertir ya.
2.  **Seguidoras (B2C)**: Mujeres que quieren aprender a bailar salsa desde el disfrute, no desde la rigidez.

## **OBJETIVO**
Crear una landing page tipo “LinkTree” para **Johanna Cueva** (Influencer & Wellness Coach), optimizada 100% para celular. Debe verse completa casi sin hacer scroll (above the fold).

**Core Philosophy**: Menos opciones = Más conversión. El único objetivo real es que las marcas o clientes potenciales **AGENDEN UNA LLAMADA** (o llenen el formulario de pre-cualificación).

## **ESTRUCTURA (3 Opciones Grandes y Claras)**

1.  **MI TRABAJO (Portfolio)**:
    *   *Microcopy*: "Resultados reales en campañas recientes."
    *   *Acción*: Redirige a una galería rápida o reel de mejores trabajos.
2.  **SERVICIOS & PRICING**:
    *   *Microcopy*: "Paquetes de colaboración y tarifas."
    *   *Acción*: Redirige al PDF/Página de precios (donde ven el valor antes de hablar).
3.  **TRABAJAR CONMIGO (Primary CTA)**:
    *   *Microcopy*: "Agenda una sesión estratégica."
    *   *Acción*: Redirige al formulario de cualificación en GoHighLevel (para filtrar clientes).

## **REQUISITOS CLAVE (UX / CONVERSIÓN)**
-   **Mobile-first absoluto**: Diseño pensado para pulgares en pantallas verticales.
-   **Sin Scroll**: Toda la información vital debe caber en la primera pantalla.
-   **Trust Signals**: Foto circular de Johanna (estilo editorial) + "Based in Miami • Wellness • Dance".
-   **Footer Mini**: Copyright + iconos ultra-minimalistas de IG/TikTok.

## **ESTÉTICA / BRANDING ("Ballet Studio Vibe")**
*Basado en la referencia visual proporcionada (Behance: Ballet Studio).*

-   **Estilo**: Minimalismo cálido, editorial, *high-end wellness*.
-   **Paleta de Colores**:
    *   **Fondo**: Cream / Beige Suave (`#F9F7F2` o `#F5F5F0`) - *Evita el blanco puro clínico.*
    *   **Texto**: Carbón Suave (`#1A1A1A`) - *Más elegante que el negro puro.*
    *   **Acento**: Terracota Muted o Rosa Viejo (`#CCA498` o `#B88B80`) para botones o hovers.
-   **Tipografía**:
    *   **Headings**: Serif elegante y editorial (ej. *Playfair Display*, *Bodoni* o similar disponible en sistema).
    *   **Cuerpo**: Sans-serif limpia y geométrica (ej. *Montserrat* o *Lato*).
-   **UI Elements**:
    *   Botones con bordes muy finos (1px) y esquinas suavemente redondeadas (pero no "pills" completos, algo más sofisticado).
    *   Sombras difusas y muy suaves ("soft glow") en lugar de drop-shadows duros.

## **TRACKING / MEDICIÓN (Sistema GoHighLevel)**
Cada clic debe ser rastreable. Implementaremos el sistema de "Páginas Puente":

1.  **Rutas Internas (GHL)**:
    *   `/go/portfolio` -> Redirige a URL del Portfolio + UTMs.
    *   `/go/services` -> Redirige a URL de Precios + UTMs.
    *   `/go/apply` -> Redirige al Formulario/Booking + UTMs.

## **ENTREGABLES TÉCNICOS**
1.  **Snippet GHL (All-in-One)**: HTML + CSS + JS en un solo bloque para pegar en "Custom HTML".
2.  **Preview Local**: Archivos `index.html`, `styles.css`, `script.js` para visualizar el diseño.
3.  **Guía de Implementación**: Paso a paso para configurar los redirects y UTMs en GoHighLevel.
