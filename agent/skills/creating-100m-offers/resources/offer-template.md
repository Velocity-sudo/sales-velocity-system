# Plantilla Oferta $100M — Gold Standard (11 Secciones)

Esta es la estructura **exacta** del documento final en Notion, verificada contra la plantilla maestra de Diego Páramo (`303e0f37-6c6d-81f1-bf4b-eeb9c41ba021`).

> [!IMPORTANT]
> El template usa SOLO estos tipos de bloque Notion:
> - `heading_1` (secciones principales, con colores específicos)
> - `paragraph` (normal, **bold**, ***bold+italic***)
> - `bulleted_list_item`
> - `divider`
> - `paragraph` vacío (separador entre sub-secciones)
>
> NO usar: `callout`, `quote`, `heading_2`, `heading_3`, `numbered_list_item`, `code`, `to_do`

> [!CAUTION]
> **REGLAS DE CONTENIDO (aplican a TODAS las secciones):**
> - ❌ PROHIBIDO tecnicismos (ROI, funnel, pipeline, LTV, CAC, churn, KPI, etc.)
> - ❌ PROHIBIDO anglicismos innecesarios
> - ✅ Escribir como si le explicaras a un amigo inteligente que NO sabe de marketing
> - ✅ Frases cortas (máximo 20 palabras por oración)
> - ✅ 1 idea = 1 bullet (específico, concreto, con números cuando sea posible)
> - ✅ La oferta debe sonar tan buena que el lector piense: "Sería tonto NO aceptar esto"
> - ✅ El nombre de la oferta debe ser corto (2-4 palabras), memorable, y funcionar en conversación casual

## Referencia: Bloques Notion API

### heading_1 con color y bold

```json
{
  "object": "block",
  "type": "heading_1",
  "heading_1": {
    "rich_text": [{
      "type": "text",
      "text": { "content": "NOMBRE DE SECCIÓN" },
      "annotations": { "bold": true, "color": "yellow_background" }
    }]
  }
}
```

### paragraph bold (sub-label)

```json
{
  "object": "block",
  "type": "paragraph",
  "paragraph": {
    "rich_text": [{
      "type": "text",
      "text": { "content": "Label:" },
      "annotations": { "bold": true }
    }]
  }
}
```

### paragraph bold+italic (resumen)

```json
{
  "object": "block",
  "type": "paragraph",
  "paragraph": {
    "rich_text": [{
      "type": "text",
      "text": { "content": "Resumen importante" },
      "annotations": { "bold": true, "italic": true }
    }]
  }
}
```

### paragraph vacío (separador)

```json
{
  "object": "block",
  "type": "paragraph",
  "paragraph": { "rich_text": [] }
}
```

### bulleted_list_item

```json
{
  "object": "block",
  "type": "bulleted_list_item",
  "bulleted_list_item": {
    "rich_text": [{
      "type": "text",
      "text": { "content": "Contenido del bullet" }
    }]
  }
}
```

### divider

```json
{
  "object": "block",
  "type": "divider",
  "divider": {}
}
```
