## **CÓMO USAR ESTE TEMPLATE**

### Step 1: Completar información del cliente
Después de crear el VSL, ya tienes toda esta información:
- Avatar, pain points, promise
- Testimonials recopilados
- Objeciones identificadas
- CTA y guarantee definidos

### Step 2: Seleccionar personalidad de marca
Basándote en el nicho del cliente, selecciona:
- Personalidad visual (Professional / Aspirational / Energetic / etc.)
- Paleta de colores correspondiente
- Tono de voz del copy

### Step 3: Completar la sección "INFORMACIÓN ESPECÍFICA DEL CLIENTE"
Rellena todos los placeholders con datos reales del VSL.

### Step 4: Copiar y pegar en Antigravity
Envía el prompt completo adaptado y Antigravity generará el código listo para producción.

## **EXTRAS OPCIONALES SEGÚN COMPLEJIDAD**

### Para VSLs con Webinar Registration:
Añadir al prompt:
```
ADICIONAL: Esta landing es para registro de WEBINAR en vivo.

Modificaciones necesarias:
1. Above the fold debe incluir:
   •  Contador regresivo hasta el webinar
   •  Fecha/hora específica
   •  "LIVE" badge si es en vivo

2. CTA debe ser:
   •  "Reservar Mi Plaza Gratis"
   •  Form inline (Nombre + Email)
   •  Submit → Thank you page con calendar add

3. Post-registro:
   •  Email confirmation automático
   •  Reminder sequence (24h antes, 1h antes)
   •  Replay disponible [SÍ/NO]
```

### Para Landing con Countdown Timer:
```
ADICIONAL: Incluir countdown timer funcional.

Configuración:
•  Deadline: [FECHA_HORA específica ej: "Feb 15, 2026 23:59:59 GMT-5"]
•  Acción al expirar: [OCULTAR_OFERTA / CAMBIAR_PRECIO / REDIRECT]
•  Estilo: [Flipclock / Minimal circles / Bold numbers]
•  Ubicación: [Above video / Below CTA / Sticky bottom]
```

### Para Landing con Exit-Intent Popup:
```
ADICIONAL: Exit-intent popup para retener visitantes.

Popup debe incluir:
•  Headline: [LAST_CHANCE_OFFER ej: "Espera! Antes de irte..."]
•  Oferta especial: [DISCOUNT / BONUS / FREE_CONSULTATION]
•  CTA: [CTA_TEXT ej: "Reclamar Mi Descuento"]
•  Close button obvio (no engañoso)

Trigger:
•  Mouse sale de viewport (desktop)
•  Scroll hacia atrás (mobile)
•  Después de X segundos en página
```
