# Landing Page Template for VSL

This template creates a high-converting landing page that supports your VSL (doesn't compete with it).

**Core Principle**: The page has ONE goal → Get them to watch the video and take action.

---

## **SECTION 1: ABOVE THE FOLD**

### **Eyebrow (Pre-Headline)**
Small text above the main headline. Use to:
- Qualify the audience
- Show social proof
- Create urgency

```
[Option 1: Audience Qualifier]
Atención: Para [Avatar Específico] que [Problema/Deseo]

[Option 2: Social Proof]
Más de [X número exacto] personas han logrado [Resultado]

[Option 3: Urgency]
[Tiempo limitado / Plazas limitadas]: [Beneficio]
```

### **Headline (Main Promise)**

Use ONE of these proven formulas:

**Formula 1: Promise + Unique Mechanism**
```
Descubre Cómo [Resultado Soñado] con [Tu Mecanismo Único]
(En [Tiempo] o Sin [Fricción Común])
```

**Formula 2: "How to X Without Y"**
```
Cómo [Lograr Resultado Deseado] Sin [Dolor/Objeción Principal]
```

**Formula 3: The Secret**
```
El Secreto de [Expertos/Grupo] para [Resultado] 
Que [Competencia/Industry] No Quiere Que Sepas
```

**Formula 4: Question Hook**
```
¿[Dolor Específico] Te Está Impidiendo [Resultado Deseado]?
```

### **Subheadline (Reinforcement)**
Explain the "next step" or give a hint of the method:

```
[En este video de [X] minutos, descubrirás [Mecanismo/Método] 
que [X personas] están usando para [Resultado Específico]]
```

### **Video Player**
- Embed VSL here
- NO autoplay (illegal in many places + annoying)
- Optional: Play button overlay with text "▶ Ver Video Ahora"

### **Primary CTA Button (Below Video)**

**Low-Risk CTAs** (Use estas en vez de "Comprar" o "Ordenar"):
```
✅ "Comenzar Mi [Resultado]" (Pronombre "Mi" > "Tu")
✅ "Ver Mis Opciones"
✅ "Acceder Ahora"
✅ "Sí, Quiero [Beneficio Principal]"
✅ "Obtener Acceso Instantáneo"
```

**Button Color**: High contrast with background (Test: Green, Orange, Red)

**Under Button**: Add micro-copy to reduce anxiety
```
💳 Garantía de 30 Días | ✅ Acceso Instantáneo | 🔒 Pago Seguro
```

## **SECTION 2: BENEFIT BULLETS (Optional - Si el video es corto)**

Si el VSL es < 5 minutos, refuerza con bullets debajo del video.

### **"So That" Formula for Benefits**

```
DON'T list features:
❌ "Acceso a 50 módulos de entrenamiento"

DO list benefits:
✅ "Acceso a 50 módulos de entrenamiento paso a paso 
   para que puedas dominar [habilidad] en solo [X semanas]"
```

**Template:**
```
Al unirte hoy, descubrirás:

✓ [Beneficio 1 con resultado específico]
   Para que [emotional payoff/transformation]

✓ [Beneficio 2 con resultado específico]
   Para que [emotional payoff/transformation]

✓ [Beneficio 3 con resultado específico]
   Para que [emotional payoff/transformation]

✓ [Beneficio 4 con resultado específico]
   Para que [emotional payoff/transformation]

✓ [Beneficio 5 con resultado específico]
   Para que [emotional payoff/transformation]
```

Keep it to 3-7 bullets max (brain processes lists better).

## **SECTION 3: SOCIAL PROOF**

### **Testimonials**

**Strategic Placement**: Use testimonials to handle objections.

```
[PHOTO of person]
"[Specific result with numbers]"
- [Name], [Title/Industry]

Example:
"Pasé de $3,000/mes a $47,000/mes en solo 90 días usando este método exacto."
- María González, Coach de Negocios
```

**Best Practices:**
- Use real photos (stock photos kill trust)
- Specific results > vague praise
- Testimonials that pre-empt doubts:
  - "Pensé que era muy caro, pero se pagó solo en 2 semanas"
  - "Soy terrible con la tecnología, pero esto fue super fácil"

### **Logos (If Applicable)**

```
Como se vio en: [Media logos]
Empresas que confían en nosotros: [Client logos]
```

### **Social Proof Numbers**

```
✓ [9,984] clientes satisfechos (Use exact numbers, not rounded)
✓ [4.9/5] estrellas basado en [1,247] reseñas
✓ Más de [$X millones] generados para nuestros clientes
```

## **DESIGN BEST PRACTICES**

### **Layout:**
- **Single Column** (no distractions)
- **No Navigation Menu** (removes exit points)
- **No External Links** (keep them on page)
- **White Space** (let content "breathe")
- **Mobile Responsive** (60%+ traffic is mobile)

### **Colors:**
- **High Contrast**: Dark text on light background or vice versa
- **CTA Button**: Contrasting color that pops
- **Consistent Branding**: Match brand colors

### **Typography:**
- **Headline**: Larger, bold, attention-grabbing
- **Body**: 16-18px minimum (readability)
- **Line Height**: 1.5-1.6 for body text
- **Font**: Sans-serif for web (easier to read)

### **Above the Fold:**
Must include:
1. Headline
2. Subheadline  
3. Video player
4. CTA button

All visible without scrolling.

## **MESSAGE MATCH CHECKLIST**

Critical for paid traffic:

```
Ad Promise → Landing Headline → VSL Hook

Example:
❌ BAD:
- Ad: "Pierde peso rápido"
- Landing: "Bienvenido a mi clínica"

✅ GOOD:
- Ad: "Pierde peso sin ejercicio"
- Landing: "Cómo perder peso sin ejercicio ni dietas extremas"
- VSL Hook: "Si estás cansado de ejercicio que no funciona..."
```

## **A/B TESTING VARIANTS**

Test these elements in this order (biggest impact first):

1. **Headline** (3-5 variants)
2. **CTA Button Copy** (3+ variants)
3. **CTA Button Color** (Green vs Orange vs Red)
4. **Guarantee Placement** (Above vs Below CTA)
5. **Video Thumbnail** (If not autoplay)
6. **Bullet Points** (Benefits vs Features)

**Tool Recommendation**: Use Google Optimize, Unbounce, or Instapage for split testing.

## **MOBILE OPTIMIZATION**

60%+ of traffic is mobile. Ensure:

- Video player is responsive
- CTA buttons are thumb-friendly (min 44x44px)
- Text is readable without zooming (16px minimum)
- No horizontal scrolling
- Fast load time on 4G
