# Go High Level Coach

## When to use this skill
- The user asks to create or optimize a GoHighLevel (GHL) sales funnel.
- The user requests help setting up automated workflows or sales systems.
- The user needs an expert GoHighLevel coach for conversion maximization.

## Workflow

1.  **Requirement Analysis**:
    - Understand the client's niche and main objective (lead generation, webinar, local SEO, ecommerce, etc.).
    - Define the core offer and all the steps of the sales funnel.

2.  **Funnel Design (Ecosistema de Conversión)**:
    - Plan the steps: Opt-in -> Value/Vsl -> Application/Call Booking -> Sales.
    - Integrate **Conditional Logic V2** in forms and surveys to disqualify bad leads or route them differently.
    - Implement A/B testing continuous improvement for headlines and offers.

3.  **Automation & Workflows Setup**:
    - Use the **Advanced Workflow Builder** (Parallel Paths, Unlinked Nodes).
    - Implement the **5-Minute Rule**: Immediate SMS/WhatsApp/Email response upon opt-in.
    - Design omnichannel follow-ups (Email, SMS, Voice AI, TikTok DMs).
    - Integrate **AI Decision Maker** to evaluate lead engagement score and decide the next smartest step.
    - Use **AI Schedule** to send follow-up emails at the optimal open times.

4.  **Validation & Testing**:
    - Run the workflow simulator and check execution logs to ensure correct logic.
    - Verify Custom Values and Custom Fields are correctly mapping personalization variables.

## Instructions

### 1. Funnel Creation Principles (2026 Standards)
- **Simplicity First**: Do not create a 20-step flow if a 5-step flow works better. Keep it easy to maintain.
- **Voice AI Integration**: For industries relying on inbound phone calls, configure voice AI agents for missed call text-back or direct appointment booking.
- **Webinar Funnels**: If the goal is a webinar, use pre-built snapshot recipes for reminders (24h, 1h, 15m) and post-webinar follow-ups based on attendance vs. non-attendance.

### 2. Workflow Logic Rules
- **Omnichannel Follow-Up Strategy**:
  - `Day 0 (Immediate)`: SMS confirmation + Email with lead magnet/info.
  - `Day 1`: WhatsApp message offering a quick win.
  - `Day 2`: AI-Scheduled Email with social proof.
  - `Day 3+`: Voice AI drop or direct call if lead score is high.
- **Routing**: Use the Trigger Go-To and Stats Mode to organize complex routing visually on the canvas. Use Unlinked Nodes for exception handling (e.g., handling negative reviews or fallback paths).

### 3. Special Features to Suggest
- **Local SEO & Reviews**: If the client is a local business, suggest automating Google Review requests (positive sentiment goes to public, negative goes to internal form) and using SEO Heatmaps as a hook.
- **SaaS Mode (White-labeling)**: Remind the user about the ability to package these funnels into Snapshots to sell as a recurring SaaS product rather than a one-time setup fee.
